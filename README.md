# SecurBot

SecurBot is a console chatbot project made in C#. It is about online safety and cyber facts, with menus, profiles and password system. The app start by playing audio greeting, then it ask user to sign in with icon and password. After sign in you can see menu with options like save profile, load, change password, delete, and sign out. Some spelling and mistakes are still here but it works fine.

## How to Run
1. Clone or download the repo.
2. Open the solution in Visual Studio (or VS Code with .NET SDK).
3. Make sure you have .NET 8 installed.
4. Build the project (`dotnet build`).
5. Run the program (`dotnet run`).
6. Listen to the audio greeting, then sign in with your profile name, icon (@, #, $, %) and password.
